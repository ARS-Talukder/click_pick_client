# рџ›’ Click Pick

Click Pick is a modern, responsive, and user-friendly eCommerce web application built with **React.js**, **Node.js**, **Express.js**, and **MongoDB**. It provides a seamless shopping experience where users can browse products, view product details, add items to their cart, and place orders. The application also includes a secure admin dashboard for managing products and orders.
